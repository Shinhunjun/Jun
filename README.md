# Hunjun Shin — Portfolio

Next.js portfolio for [shinhunjun.github.io/Jun](https://shinhunjun.github.io/Jun/).

Source lives in [`portfolio-next/`](./portfolio-next). Deployed via GitHub Actions on push to `main`.

```bash
cd portfolio-next
npm install
npm run dev    # http://localhost:3000
npm run build  # static export → portfolio-next/out
```
